# 🎨 DO IT LIKE A DREAMER — GIMP MCP

> *Follow your heart and do it like a dreamer.*

**One prompt. One canvas. Zero manual edits. 11,000 RPM.**

![Do It Like A Dreamer — a dreamer's dashboard rendered entirely through GIMP MCP](https://raw.githubusercontent.com/MushroomFleet/do-it-like-a-dreamer-GIMP-MCP/refs/heads/main/do-it-like-a-dreamer.png)

**Do It Like A Dreamer (GIMP MCP)** is the desktop-editor counterpart to the [web experiment of the same name](https://github.com/MushroomFleet/do-it-like-a-dreamer-html). Every pixel in the image above was painted by an AI assistant driving **GIMP 3.2** — no mouse, no menus, no manual retouching. A night-to-dawn sky, a glowing horizon, a starfield, and a tachometer pinned deep into the redline were all composed from a single natural-language brief, one tool call at a time.

It exists to prove a point: with the [Model Context Protocol](https://modelcontextprotocol.io) you can put a full, dependency-heavy raster editor under an assistant's hands and have it *compose* — not just filter — a finished, informational image. **The picture is the product, and the tool calls are the documentation.**

---

## ✨ What's in the image

- 🌌 **A procedural night→dawn sky** — a seven-stop gradient painted band-by-band in GIMP's Python-Fu (the high-level `gradient_fill` is a no-op on GIMP 3.2, so the gradient was generated by hand), plus a 200-star field scattered across the upper sky.
- 🌅 **A soft radial horizon glow + a blue "dream-moon"** — real foreground→transparent radial gradients riding on **Screen-blend** layers, so light *adds* over the sky instead of covering it.
- 🔴 **A tachometer that goes to eleven** — built entirely from polygon, arc, and ellipse selections: a metallic bezel, green / amber / red zones, twenty-five tick marks, a tapered needle **pinned at 11,000 RPM deep in the redline**, and a hub.
- 🔢 **A digital readout** — `×1000 r/min` scale and a glowing `11,000` on the dial face.
- 🔤 **Native GIMP text layers** — perfectly-centered gauge numbers, the title, the *"do it like a dreamer"* tagline, three info chips (`GIMP 3.2.4 · 80 MCP TOOLS · 59/59 TESTS`), and the footer.

## 🔧 How it was made

| Piece | Role |
|---|---|
| [**GIMP 3.2**](https://www.gimp.org/) | The renderer — the actual desktop app doing the drawing |
| [**maorcc/gimp-mcp**](https://github.com/maorcc/gimp-mcp) | The MCP server that bridges GIMP's Python-Fu API to the Model Context Protocol |
| **An MCP client** (Claude Code) | Drove the ~80 GIMP tools, and dropped into raw `call_api` Python-Fu where the built-in tools couldn't reach |
| [**`prompt.md`**](https://github.com/MushroomFleet/do-it-like-a-dreamer-GIMP-MCP/blob/main/prompt.md) | The one-shot creative brief |

Techniques used: procedural gradients, Screen / additive compositing layers, polygon selections for the gauge geometry, native `Gimp.TextLayer` typography, and a final flatten + PNG export — all issued as MCP tool calls, none of it clicked by hand.

## 🚀 Reproduce it

1. Install and run the MCP server from [**maorcc/gimp-mcp**](https://github.com/maorcc/gimp-mcp), then start it inside GIMP via **Tools ▸ MCP ▸ Start MCP Server** (it listens on `localhost:9877`).
2. Point your MCP client (Claude Desktop / Claude Code / any MCP host) at the server.
3. Hand it [`prompt.md`](https://github.com/MushroomFleet/do-it-like-a-dreamer-GIMP-MCP/blob/main/prompt.md) and let it drive.

Your result won't be pixel-identical — that's the point. *Follow your heart.*
